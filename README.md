An e-commerce application that allows users to purchase soccer jerseys and footwear. The app is built with Next.js on the front-end and Django on the back-end, providing a robust and scalable platform for online transactions.

In addition to traditional search functionality, the app also features advanced search options that enable users to browse jerseys by player or nationality. This helps users to quickly find and purchase the specific items they are looking for.

Overall, the app is designed to provide users with a seamless and convenient shopping experience, and is equipped with features such as payment processing using stripe.

run command  python manage.py runserver --settings=backend.settings.dev
